How to Use

Engaged Social Menu is super simple to implement. There are 4 files that must be included, for the menu to work perfectly. These are;

  • Engaged.css
  • Font-awesome.css
  • Engaged.js
  • jQuery.js
Demo is included with with other files. Two file named as demo.css and demo.js are optional and can be removed if user don't want it.

Demo contain four different navigation menus. You may choose your own style and alignment, or even the full width on top. User may customize the menu but it is important not to remove the call to function engaged. Otherwise there will be no animation.
